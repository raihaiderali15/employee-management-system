# Employee Management System (EMS)

The **Employee Management System (EMS)** is a web-based application designed to manage employee tasks efficiently. It allows administrators to assign tasks to employees, while employees can log in to view their assigned tasks and update their work status. This project demonstrates **role-based access control** and **frontend state management** using React.
